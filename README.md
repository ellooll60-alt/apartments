# 🏢 منصة إدارة الشقق المفروشة – The Golden Deal

منصة احترافية لإدارة الشقق المفروشة والحجوزات، مبنية باستخدام **Streamlit** و **Supabase**، مع نظام صلاحيات كامل (أدمن – مدير – موظف)، وتقويم تفاعلي، وجداول قابلة للبحث، ولوحة تحكم متقدمة.

---

## 🚀 المميزات الرئيسية

### 🔐 نظام صلاحيات كامل (Role-Based Access)
- صفحة دخول آمنة
- إعادة توجيه تلقائي حسب الدور:
  - Admin
  - Manager
  - Employee

### 📅 تقويم حجوزات تفاعلي (FullCalendar)
- عرض الحجوزات على شكل Calendar
- إضافة حجز عند الضغط على يوم
- تعديل حجز عند الضغط على حدث
- يدعم JSON.parse لعرض البيانات بدون مشاكل

### 📋 جدول حجوزات تفاعلي قابل للبحث
- بحث بالاسم – رقم الوحدة – المنصة
- فرز تلقائي
- عرض كل الأعمدة المهمة
- متوافق مع Streamlit Cloud

### 📊 لوحة تحكم احترافية
- إحصائيات عامة
- متابعة يومية – أسبوعية – شهرية
- تقارير شهرية وسنوية
- توقعات مستقبلية (Forecasting)

### 🧱 Tabs علوية قابلة للتمرير
تشمل:

1. الإحصائيات العامة  
2. التقويم التفاعلي

### 🗄️ قاعدة البيانات – Supabase
الجداول الأساسية:

- `bookings`
- `units_names`
- `clients`
- `platforms`
- `expenses`
- `compensations`
- `users`
